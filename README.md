# WebGPU Arena Shooter

Browser-based arena shooter foundation built with Vite, React, TypeScript, React Three Fiber, and a WebGPU-first Three.js renderer path.

## Current scope

- Minimal arena-room renderer foundation
- WebGPU-first renderer with a safe fallback path
- Shared renderer status store with Zustand
- Small in-scene debug overlay for renderer mode, initialization state, and FPS

## Commands

- `npm run dev`
- `npm run build`
- `npm run lint`
